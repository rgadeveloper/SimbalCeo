Estructura del Archivo a Cargar

Esta funcionalidad permite al usuario realizar la carga masiva en el simbal de los amarres/desamarres de Macros Virtuales a Barrio(s)/Circuito(s).

El archivo debe tener la siguiente estructura:

IDMACROVIRTUAL;IDPADREAMARRE;PERIODO;TIPO

El campo ‘TIPO’ se refiere al tipo de Zona a asociar:

  • Asociar a Barrio. TIPO = B
  • Asociar a Circuito. TIPO = C

Ejemplo:

MVE061212;1104;201208;C

MVG061212;95;201208;B
